ZIM - Charite Wiki Generator

Verwirrendes Python Konstrukt um ZimWiki für Charite Lernziele zu nutzen.

Mit diesem Programm können Lernziele, welche man als xls Datei von der LLP herunterladen kann, in eine ZimWiki Datenbank umgewandelt werden.
Hierzu wird ein neues Wiki generiert, welches ihr danach in andere hineinfügen könnt.

Diese Version wurde mit den Inhalten des 5. Semesters getestet.
